# 🏠 내 홈페이지 설명서

> 이 파일을 AI에게 주면, AI가 나만의 홈페이지를 만들어줘요! 빈칸을 꼼꼼히 채울수록 더 멋진 홈페이지가 만들어져요 ✨

---
## 역할(페르소나)
너는 Next.js 웹 개발자야. 초등학생이 이해할 수 있는 멋진 홈페이지를 만들어줘.

---
## 👤 1. 나는 누구?

- **이름 (또는 닉네임):** 쏠라 / 닉네임: 쏠랭이
- **나이 ** 1살
- **사는 곳 (도시 정도만):** 김포
- **좋아하는 것들:** 산책, 가족과 함께 있기, 간식, 터그놀이, 털 빗기
- **싫어하는 것들:** 비, 혼자 있기, 밥 먹기, 잠 깨우기
- **장래희망:** 훌륭한 개

---

## 🎯 2. 이 홈페이지는 어떤 곳이에요?

**홈페이지 소개:** 쏠랭이가 좋아하는 것들을 모아놓은 쏠라 만의 비밀기지!

---

## 🎨 3. 어떤 느낌으로 만들고 싶어요?

**분위기 선택:**

- [ ] 밝고 귀엽게 🌸
- [ ] 깔끔하고 심플하게 🤍
- [ ] 자연스럽고 여름엔 시원하고 겨울엔 따뜻하게

**좋아하는 색깔:** 노란색, 연분홍색, 하늘색, 하얀색

**싫어하는 색깔:** 어두운 색

**참고하고 싶은 느낌:** 강아지 운동장 느낌

---

## 📄 4. 홈페이지에 어떤 내용이 들어갔으면 해요?

- [ ] 자기 강아지 소개 (이름, 나이, 좋아하는 것)
- [ ] 취미 소개
- [ ] 좋아하는 것 목록
- [ ] 갤러리
- [ ] 나의 꿈 / 장래희망
- [ ] 강아지를 키우는 사람에게 꼭 하고 싶은 말
- [ ] 방명록
- [ ] 직접 추가: 좋아하는 강아지 종 TOP 5 소개

---

## ⚡ 5. 특별히 넣고 싶은 기능이 있어요?

1. 마우스 커서를 쏠라로 바꾸고 싶어요
2. 배경에 꽃잎이 떨어지면 좋겠어요

## 🚀 8. 기술스택
- Next.js App Router 방식으로 만들어줘
- Tailwind CSS로 스타일 적용해줘
- 파일별로 코드를 나눠서 줘 (page.tsx, layout.tsx 등)
- 모바일에서도 잘 보이게 해줘
